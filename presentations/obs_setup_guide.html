For recording "AI Ka Raaz" YouTube series — Face + Slides + Code

Step 1: First Time Setup (One Time Only)

1.1 Open OBS

Open Applications → OBS.app

First time: It will ask to run Auto-Configuration Wizard → Click "Yes"

Select: "Optimize for Recording" (NOT streaming)

Step 2: Create 3 Scenes (Your Video Layouts)

In OBS, Scenes = different layouts you switch between while recording.

You need 3 scenes for your videos:

Scene 1: "Slides + Face" (Main scene)

🖥️
Slides + Face

For presenting HTML slides with your face in corner

┌─────────────────────────────────────────┐ │ │ │ HTML SLIDES │ │ (Browser - Fullscreen) │ │ │ │ ┌─────────┐│ │ │ YOUR ││ │ │ FACE ││ │ │ (small) ││ │ └─────────┘│ └─────────────────────────────────────────┘
A Create Scene

Bottom-left "Scenes" panel → Click + → Name it "Slides + Face"

B Add Browser/Screen

"Sources" panel → Click + → Display Capture → OK

This captures your full screen (where slides will be fullscreen in browser)

C Add Webcam

"Sources" panel → Click + → Video Capture Device → OK

Select your camera → OK

Resize it small → Drag to bottom-right corner

Right-click webcam → Transform → Edit Transform → Set size to about 320x240

Your webcam layer should be ABOVE the display capture in the Sources list (drag to reorder). Top = front.

Scene 2: "Code Demo" (Terminal)

⌨️

Step 3: Set Up Hotkeys (Switch Scenes Fast!)

Go to Settings → Hotkeys

Set these keyboard shortcuts to switch scenes WITHOUT touching the mouse:

ActionHotkeyWhen to use
Switch to "Slides + Face" F5 Presenting slides
Switch to "Code Demo" F6 Running Python scripts
Switch to "Face Only" F7 Hook, dramatic moments
Start Recording F9 Begin!
Stop Recording F10 Done!
Now you can switch between face/slides/code with ONE key press while recording!

Step 4: Recording Your Episode!

Before you hit Record:

  • Open browser with slides fullscreen (F key in slides)
  • Open Terminal with cd ~/ai_learning/season1_foundations
  • Good lighting on your face (sit facing a window or lamp)
  • Test mic — speak and check the green audio bar moves in OBS
  • Close unnecessary apps (notifications OFF!)

Step 5: Pro Tips for Better Videos

Audio is MORE important than video. Bad video + good audio = watchable. Good video + bad audio = unwatchable. Invest in a ₹500 lapel mic.
Record in parts. Don't try to record 6 minutes perfectly. Record Hook separately, then slides, then code demo. Edit together later in CapCut/iMovie.
Make terminal text BIG. Before code demos, increase terminal font size: Terminal → Settings → Font → 20pt+. Viewers watch on phones!
OBS "Studio Mode" (button at bottom) lets you preview the next scene before switching. Great for smooth transitions.
Test record 30 seconds first. Play it back. Check: Can you hear clearly? Is face visible? Is slide text readable? Fix before recording the real thing.
