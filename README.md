# AI Ka Raaz — Zero to Hero AI/LLM Series

A YouTube series teaching AI & LLMs from absolute zero. Hinglish (Hindi + English), fun thriller-style, 5-10 min videos.

## Who is this for?
Complete beginners. Any age. No coding experience needed.

## Season 2: "AI Se Kamaao" (Build & Earn)
Build real AI apps — API calls, RAG, agents, fine-tuning, deployment.

## Season 3: "AI Expert Bano" (Advanced)
BERT vs GPT, RLHF, quantization, scaling laws, vision models.

## How to Run
```bash
# Clone
git clone https://github.com/selfcompiler/ai-ka-raaz.git
cd ai-ka-raaz

# Install PyTorch
pip install torch

# Run any episode
python3 season1_foundations/ep01_ai_thinks.py
```

## Deep Dive Scripts
Want to go deeper? These standalone scripts cover each concept in detail:
- `bigram_lm.py` — Your first language model from scratch
- `tokenization_deep_dive.py` — Character, Word & BPE tokenization
- `embeddings_deep_dive.py` — Word embeddings & positional encoding
- `cosine_similarity_explained.py` — How AI measures similarity
- `self_attention_deep_dive.py` — Attention mechanism internals
- `mini_gpt.py` — Full GPT model (~10M parameters)
